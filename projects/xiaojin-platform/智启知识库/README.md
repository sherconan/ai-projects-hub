## 📝 使用说明

1. **文档存放规则**
   - 每个文档放入对应的最细粒度文件夹
   - 例如: Dify相关文档 → `开发平台/Dify/`
   - 例如: 庖丁使用手册 → `基础能力与工具/文档解析/庖丁/`

2. **支持的文档格式**
   - PDF, DOC, DOCX, TXT, MD
   - 建议使用清晰的文件命名

3. **命名建议**
   - 使用描述性文件名
   - 包含版本信息(如有)
   - 例如: `Dify快速入门v1.2.pdf`
